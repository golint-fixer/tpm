tpm
===

A Team Password Manager Client written in go.


Usage
-----

Create a configuration file with the following format:

    % cat $HOME/.tpm.yaml
    server: "https://tpm.example.com/index.php"
    username: "my-username"
    password: "my-dkj880s0sjd-password"
